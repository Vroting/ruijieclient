## git 提交解释 ##

git clone [url](url.md)
克隆一个仓库


git checkout -b [newbrance](newbrance.md) [originbranch](originbranch.md)
以originbranch为起点岔开分支。

git checkout branch
切换到分支

git commit -a
提交所有本地改变到本地仓库

git  push
提交改变到远程仓库

git pull
与远程仓库同步
